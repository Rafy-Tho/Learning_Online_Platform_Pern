The Story of JavaScript

In 1995, a programmer named Brendan Eich was hired by Netscape Communications. His task? Build a scripting language for the web browser in just 10 days. The result was a language first called "Mocha," then "LiveScript," and finally — riding the popularity wave of Java — renamed to JavaScript.

Despite the name, JavaScript and Java are about as related as a car and a carpet. The name was a marketing decision. JavaScript was built to be simple, forgiving, and embeddable directly in HTML pages.

💡
Fun Fact

JavaScript was created in 10 days. It's now used by over 98% of all websites in the world, runs on servers, mobile apps, desktop software, and even space telescopes. Not bad for 10 days of work.

What Does JavaScript Actually Do?

Think of a website as a human body. If HTML is the skeleton (structure), and CSS is the skin and clothes (appearance), then JavaScript is the muscles and brain — it makes things move and respond.

🦴
HTML
Structure and content. The skeleton. Headings, paragraphs, buttons, images.
🎨
CSS
Visual style. Colors, fonts, layout, animations. How it looks.
⚡
JavaScript
Behavior and interactivity. Clicks, forms, data, real-time updates.

Your First Glimpse of JavaScript

Don't worry about understanding every word yet. Just read through this and get a feel for what JavaScript looks like.

script.js 
// This is a comment - JavaScript ignores it
// It's just a note for humans reading the code

// Display a message in the browser console
console.log("Hello, World!");

// Store a value in a variable
let userName = "Alex";

// Combine text with a variable
console.log("Welcome, " + userName + "!");

// Make a decision
if (userName === "Alex") {
  console.log("Hey Alex! Ready to learn JavaScript?");
} else {
  console.log("Welcome, new learner!");
}
▶ OUTPUT
Hello, World!
Welcome, Alex!
Hey Alex! Ready to learn JavaScript?

Notice how JavaScript reads almost like plain English? That's intentional. It was designed to be approachable. You'll be writing code like this — and much more — by the end of this course.

Where Does JavaScript Run?

JavaScript originally ran only inside web browsers. Today it runs virtually everywhere:

🌐
In the Browser (Client-side)

The classic home of JS. Chrome, Firefox, Safari, Edge — every modern browser has a built-in JavaScript engine. Chrome uses V8, Firefox uses SpiderMonkey.

🖥️
On the Server (Node.js)

In 2009, Ryan Dahl took Chrome's V8 engine and built Node.js — allowing JavaScript to run on servers. Now JS can power entire backends.

📱
Mobile Apps

Frameworks like React Native and Ionic let you build iOS and Android apps with JavaScript.

🖱️
Desktop Apps

Electron.js powers VS Code, Slack, and Figma — all built with JavaScript running as a desktop application.
